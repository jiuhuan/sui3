tag

Tag 是用来显示统一类型的不同选择，类似checkbox或者radio。它可以作为纯信息展示，也可以通过内部的 checkbox/radio 来实现数据操作。

Tag 分为三种用法：展示信息（不可操作），可删除，可勾选。

注意，tag组件本身不存储任何数据，数据都是通过内部的 checkbox/radio 来保存的。
